# reports

This folder is for generated runtime surfaces.

Examples:

- `operator_latest.md`
- `supervisor_latest.md`
- `governance_latest.md`
- cycle reports
- load-test reports

For public packaging, treat these as generated artifacts, not as primary source files.

## Public Rule

- keep this folder in the repository for structure
- do not rely on generated report files as the public source of truth
- regenerate runtime surfaces locally when needed
