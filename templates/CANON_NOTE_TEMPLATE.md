# canon-note-slug

- promoted_from: `wiki_lite/WIKI/...`
- promoted_at: `YYYY-MM-DD HH:MM:SS`
- claim_state: `fact | inference | opinion | hypothesis`
- evidence: `evidence path or source location`
- evidence_mode: `explicit | listed | trace`
- freshness: `YYYY-MM-DD | range`
- confidence: `low | medium | high`
- confidence_basis: `explicit | source_count | conservative_default`
- scope: `valid scope and exceptions`
- supersession: `none | target/path.md`
- stale_flag: `false | true`
- conflict_with: `none | target/path.md`
- source_count: `1+ | unknown`
- source_count_basis: `explicit | evidence_refs | trace_only | unknown`
- last_reviewed: `YYYY-MM-DD`
- temporal_state: `current | stale-sensitive | archive`

## canon

- Write the formal content worth keeping long term.

## evidence

- Record which evidence this note was built from.

## supersession

- If this note replaces an older note, record the target note path.
