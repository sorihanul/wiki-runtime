# raw-note-slug

- captured_at: `YYYY-MM-DD HH:MM:SS`
- source_scope: `inside | outside`
- source_path: `./relative/path/or/external/source`
- intended_axis: `memory | governance | skill | coordination`
- initial_judgment: `adopt | already-covered | hold | reject`

## raw

- Capture the source core without distorting it.

## caution

- Record time sensitivity, exaggeration risk, and points that still need verification.
